# öly – Beauty E-commerce Platform

A full-stack beauty products shopping website built with:

- **Frontend**: React, React Router, Axios
- **Backend**: Node.js, Express, MongoDB (Mongoose)
- **Features**:
  - Product listing & detail pages
  - Cart & checkout with address input
  - User authentication (buyer/seller/admin)
  - Admin dashboard (products + orders management)
  - AI beauty assistant chatbot

## Tech Stack

- Frontend: React 18, React Router v6
- Backend: Node.js, Express, JWT auth
- Database: MongoDB Atlas
- File upload: Multer
- Styling: Inline CSS + some component libraries
